# EstratégIA Landing Page

Landing page premium para a plataforma EstratégIA, por Rebeca Geller.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Configuração

- Checkout: `src/content.ts`, constante `CHECKOUT_URL`
- VSL: `src/content.ts`, constante `VSL_URL`
- Fotos: `src/assets/rebeca-hero.jpg` e `src/assets/rebeca-trajetoria.jpg`
